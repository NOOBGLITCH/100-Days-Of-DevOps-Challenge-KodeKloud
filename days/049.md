# Create a Kubernetes Deployment

The Nautilus DevOps team needed to deploy an Nginx application using a Kubernetes Deployment. We created a Deployment named **`nginx`** using the **`nginx:latest`** image and verified that the Deployment and its Pod were running successfully.

## Requirements

* Create a Deployment named **`nginx`**
* Use image **`nginx:latest`**
* Verify the Deployment and Pod

---

# Steps We Performed

## 1. Verify Cluster Connectivity

```bash
kubectl get nodes
```

Expected output:

```text
NAME           STATUS   ROLES           AGE
controlplane   Ready    control-plane   ...
```

---

## 2. Create the Deployment

Using the imperative command:

```bash
kubectl create deployment nginx --image=nginx:latest
```

Expected output:

```text
deployment.apps/nginx created
```

---

## 3. Verify the Deployment

```bash
kubectl get deployments
```

Expected output:

```text
NAME    READY   UP-TO-DATE   AVAILABLE   AGE
nginx   1/1     1            1           10s
```

---

## 4. Verify the Pod

```bash
kubectl get pods
```

Expected output:

```text
NAME                     READY   STATUS    RESTARTS   AGE
nginx-xxxxxxxxxx-xxxxx   1/1     Running   0          10s
```

---

## 5. Describe the Deployment

```bash
kubectl describe deployment nginx
```

Verify that the image is:

```text
Image: nginx:latest
```

---

## Quick Commands

```bash
kubectl create deployment nginx --image=nginx:latest

kubectl get deployments

kubectl get pods

kubectl describe deployment nginx
```

---

# Good to Know?

### Deployment

* Manages Pods declaratively.
* Automatically recreates Pods if they fail.
* Supports rolling updates and rollbacks.

### ReplicaSet

* Created automatically by a Deployment.
* Ensures the desired number of Pods are running.

### Pod

* Smallest deployable unit in Kubernetes.
* Created and managed by the Deployment.

### Image Tag

* `nginx:latest` explicitly uses the latest Nginx image.
* Specifying the tag ensures the required image version is used.

### kubectl create deployment

* Creates a Deployment imperatively.
* Automatically creates a ReplicaSet and Pod.
