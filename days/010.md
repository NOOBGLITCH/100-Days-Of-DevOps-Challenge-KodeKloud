# Day 10: Website Media Backup Script

## Overview

Automate media backup from App Server 3 to Nautilus Backup Server using a bash script with SSH key authentication.

## Configuration

| Parameter | Value |
|-----------|-------|
| Source | `/var/www/html/media` |
| Archive | `/backup/xfusioncorp_media.zip` |
| Remote Host | `stbkp01.stratos.xfusioncorp.com` |
| Remote User | `clint` |
| Script Path | `/scripts/media_backup.sh` |

## Prerequisites

1. **Install zip package** (run as admin):
   ```bash
   # CentOS/RHEL
   sudo yum install zip -y
   # Debian/Ubuntu
   sudo apt-get install zip -y
   ```

2. **Create local backup directory**:
   ```bash
   mkdir -p /backup
   chmod 755 /backup
   ```

3. **Configure passwordless SSH** (run as script user):
   ```bash
   ssh-keygen -t rsa -b 2048
   ssh-copy-id clint@stbkp01.stratos.xfusioncorp.com
   ```

4. **Ensure remote backup directory exists** on `stbkp01.stratos.xfusioncorp.com`:
   ```bash
   mkdir -p /backup
   chmod 755 /backup
   ```

## Script

Create `/scripts/media_backup.sh`:

```bash
#!/bin/bash
# Archives /var/www/html/media and copies to remote backup server

set -euo pipefail

SOURCE_DIR="/var/www/html/media"
ARCHIVE_NAME="xfusioncorp_media.zip"
LOCAL_BACKUP_DIR="/backup"
REMOTE_USER="clint"
REMOTE_HOST="stbkp01.stratos.xfusioncorp.com"
REMOTE_BACKUP_DIR="/backup"

# Create local backup directory if missing
[ -d "$LOCAL_BACKUP_DIR" ] || mkdir -p "$LOCAL_BACKUP_DIR"

# Verify source exists
[ -d "$SOURCE_DIR" ] || { echo "Source $SOURCE_DIR not found" >&2; exit 3; }

ARCHIVE_PATH="$LOCAL_BACKUP_DIR/$ARCHIVE_NAME"

# Create archive and copy to remote server
zip -r "$ARCHIVE_PATH" "$SOURCE_DIR"
scp -q "$ARCHIVE_PATH" "$REMOTE_USER@$REMOTE_HOST:$REMOTE_BACKUP_DIR/"

echo "Backup complete: $ARCHIVE_PATH → $REMOTE_HOST:/backup/"
exit 0
```

Make executable:
```bash
chmod +x /scripts/media_backup.sh
```

## Execution

Run as the user with SSH keys configured (no sudo):
```bash
/scripts/media_backup.sh
```

## Good to Know?

### Bash Scripting Best Practices

- **Shebang**: Always start with `#!/bin/bash` or `#!/bin/sh`
- **Error Handling**: Use `set -e` to exit on errors
- **Variables**: Quote variables: `"$variable"` to prevent word splitting
- **Functions**: Break complex logic into reusable functions

### Backup Strategies

- **Full Backup**: Complete copy of all data
- **Incremental**: Only changed files since last backup
- **Differential**: Changed files since last full backup
- **Compression**: Use zip, tar.gz to save space

### Archive Commands

- **zip**: `zip -r archive.zip directory/` (cross-platform)
- **tar**: `tar -czf archive.tar.gz directory/` (Unix standard)
- **Exclusions**: Skip temporary files and logs
- **Verification**: Test archives before relying on them

### Remote Copy Methods

- **scp**: Secure copy over SSH
- **rsync**: Efficient synchronization tool
- **sftp**: Interactive file transfer
- **Authentication**: Use SSH keys for automation
