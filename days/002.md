# 📚 Good to Know

## User Account Expiry

* **Purpose:** Automatically disable accounts after a specified date
* **Format:** `YYYY-MM-DD`
* **Check Expiry:** `chage -l username`
* **Extend Expiry:**

```bash id="t4l8xs"
sudo chage -E 2024-12-31 username
```

---

## Temporary Account Management

* Always set expiry dates for temporary accounts
* Use `chage -l` to monitor account status
* Expired accounts remain on the system but cannot log in
* Remove unused accounts with:

```bash id="m1d7qp"
sudo userdel -r username
```
